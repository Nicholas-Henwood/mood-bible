Step 1 of the Play Store process

Icon Generator

This page draws your app icons on a canvas and lets you download them as PNG files. You need exactly three files before you can run the app as a PWA or submit it to the Play Store.

Where to save these files:
Create a folder called icons next to your HTML files, then save each PNG inside it:
icons/icon-192.png
icons/icon-512.png
icons/icon-maskable-512.png
  1. Click each Download button below and save the file.
  2. Create an icons folder inside your project folder (C:\Youtube\icons\).
  3. Move all three downloaded PNG files into that icons folder.
  4. Then proceed to the Play Store Setup Guide.

192 × 192 — Standard Icon

Used as the home-screen icon on most Android devices.

512 × 512 — Standard Icon

Required by Google Play and used at full resolution on large-screen devices.

512 × 512 — Maskable Icon

Android's adaptive icon system crops icons into circles, squircles, etc. This version adds safe-zone padding so the cross is never clipped.
